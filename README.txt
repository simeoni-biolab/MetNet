CONTENTS OF THE MetNetSourceFiles FOLDER

The downloaed folder is composed of the following files:
- README.txt : this file
- lib : folder of software libraries used by MetNet 
- src : source files folder
- organismList.txt : configuration file containing the list of organisms available in KEGG
- pathwayList.txt : configuration file containing the list of patways to be considered for the comparison
